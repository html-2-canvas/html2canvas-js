Installation Methods

Choose the installation method that works best for your project

1

NPM Installation

Install using Node Package Manager for seamless integration with your build tools.

npm install html2canvas
2

Yarn Installation

Prefer Yarn? No problem! Install the package using Yarn package manager.

yarn add html2canvas
3

CDN Link

For quick prototyping or simple projects, use the CDN link directly in your HTML.

<script src="https://cdn.jsdelivr.net/npm/html2canvas@latest"></script>

Quick Start Guide

Get up and running in three simple steps

Basic Usage Example

import html2canvas from 'html2canvas'; // Capture a specific element html2canvas(document.querySelector('#capture')) .then(canvas => { document.body.appendChild(canvas); });

That's it! The element will be rendered as a canvas and appended to your page. For more examples and advanced usage, visit our homepage.
